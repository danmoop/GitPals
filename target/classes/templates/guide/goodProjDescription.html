How to create good description for your project

When you are about to submit your project, you should check twice that description describes it the best.

  • 1. Make sure you have chosen all the requirements (skills) for the project you really need
  • 2. Describe how much time you will be spending on this project.
  • 3. Provide your Email, Skype, Discord or any other service where you will be discussing everything about the project.
  • 4. Provide deadline for your project
  • 5. Provide any working concept as an example (if you have, of course)
  • 6. Mention if this is a profitable or non-profitable project.

Good luck to you!
